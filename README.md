# tailwind.d

[![tailwind.d - Docs](https://img.shields.io/badge/tailwind.d-Docs-blue)](https://rorkh.github.io/tailwind-d/tailwind.html)

Tailwind for D lang.

# Prequisites

To use Tailwind.d you need to install Tailwind CLI.

```sh
npm install -D tailwindcss
```

# Installation

```sh
dub add tailwind-d
```
